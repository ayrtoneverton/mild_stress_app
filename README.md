# Mild Stress App

Trata-se de um mini-game de raciocínio logico e enigmas(puzzles) desenvolvido em React Native.


## Pré-requisito

https://nodejs.org/en/download/


## Iniciando

Após clonar e estar na raiz do projeto:
```shell
$ npm install
$ npm start
```

Em outro terminal:
```shell
$ npm run android
```
